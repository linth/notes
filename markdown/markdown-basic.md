# Markdown 基本語法

## 1. 標題
```
# This is an H1
## This is an H2
### This is an H3 ######
#### This is an H4 ####
##### This is an H5 #####
###### This is an H6 ######
```
# This is an H1
## This is an H2
### This is an H3 ######
#### This is an H4 ####
##### This is an H5 #####
###### This is an H6 ######

## 2. 字體樣式
### 粗體
```
**bold**
```
**bold**

### 斜體
```
_bold_
*bold*
```
_bold_
*bold*

### 斜體 + 粗體
***test***
```
***test***
```
***test***

### 刪除線
```
~~your text~~
```
~~your text~~

### 顏色
```
<font color="red">1111</font>
```
<font color="red">1111</font>

## 3. 內容項目
### 水平線
```
---
```
---

### 階層
```
> 階層1
>> 階層2
>>> 階層3
```
> 階層1
>> 階層2
>>> 階層3

### 小區塊
```
`Format one word or one line*`
```
`Format one word or one line*`

### 大區塊
```python

  pass
```
